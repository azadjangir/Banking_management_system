**Banking Management System**



**Project Overview** 



This project is a simple Banking Mangement system developed using python using the concepts of OOPs the application allows users to manage bank accounts , add money and with draw it , saving the data in a CSV file allowing the data to persist between program executions





**Features**



The system provides the following features:



* Create a new bank account
* Deposit money into an account
* Withdraw money from an account
* View all existing accounts
* Delete an account
* Save account information to a CSV file
* Load saved account information from a CSV file
* Menu driven interface
* Basic exception handling for file operations





**Installation:**

Download  the project.

Open the project folder in Visual Studio Code.

Make sure all Python files are located in the same folder.

Ensure that accounts.csv exists in the project directory.



**Running the Program:**


Run the file Main.py in terminal so that the banking Management system menu appears 



**Example Usage:**

Select Create Account.

Enter the account name, customer name, and starting balance.

Deposit or withdraw money.

View all accounts.

Save the accounts before exiting.

Load the saved accounts when restarting the application.
